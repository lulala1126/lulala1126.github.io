变量定义

read-only variables are defined using the keyword val. They can be assigned a value only once.

Variables that can be reassigned use the var keyword.

形参带缺省值的函数

当子类继承父类并重写其方法时，缺省值不能改变。

在调用函数时，可以通过形参名称赋值，减少不必要的赋值操作

Infix notation

infix函数使用infix关键字表示,调用时可省略点以及括号，但必须满足下列要求:

  • 必须是成员函数或者扩展函数
  • 只能有一个形参
  • 形参不能为可变参数（Varargs)及不能有缺省值

infix函数的优先级低于算术操作符（+-*/）高于逻辑操作符（&&、||）
